Edge Cases

This page deliberately tests robustness of the OpenGraph parser.

  • og:type — absent (should be null)
  • og:description — absent (should be null)
  • og:site_name — blank string (should be null after isBlank filter)
  • og:locale — whitespace only (should be null after isBlank filter)
  • og:image:width / height — absent despite og:image being present
  • og:title — duplicated, first value wins
  • article:* — present even though og:type ≠ article
  • non-OG meta tags — present, should be ignored
